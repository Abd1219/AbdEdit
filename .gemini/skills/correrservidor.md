# Skill: correrservidor

Esta skill inicia un servidor local para probar la aplicación web AbdEdit.

## Comandos
```powershell
npm run correrservidor
```

## Descripción
Utiliza `http-server` via `npx` para servir los archivos estáticos en el puerto por defecto (normalmente 8080). Esto permite probar las funcionalidades de PWA, el Service Worker y el manifiesto.
